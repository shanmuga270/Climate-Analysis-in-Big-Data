# Climate Analysis in Big Data

## Overview

Climate Analysis in Big Data is a project that processes and analyzes
large-scale climate datasets using big data technologies. The system
cleans, analyzes, visualizes, and predicts climate trends to help
understand long-term environmental changes.

## Features

-   Climate data preprocessing and cleaning
-   Large-scale data analysis
-   Data visualization
-   Climate trend prediction
-   Distributed data processing

## Technologies Used

-   Python
-   Apache Hadoop
-   Apache Spark
-   PySpark
-   Pandas
-   Matplotlib
-   Scikit-learn
